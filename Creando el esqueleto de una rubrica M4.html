Taller Guiado: Creando el Esqueleto de una Rúbrica

Completa esta plantilla para diseñar la estructura base de una rúbrica para una tarea simple.

Paso 1: Define el Objetivo de Aprendizaje

¿Qué quieres que el estudiante sea capaz de hacer después de completar la tarea? Sé específico.

Paso 2: Lista 3 Criterios de Evaluación

¿Cuáles son los 3 aspectos más importantes que evaluarás en el párrafo argumentativo?

Paso 3: Define una Escala de 3 Niveles

Nombra los 3 niveles de desempeño, desde el más alto hasta el más bajo.

¡Aquí está tu Esqueleto de Rúbrica!
